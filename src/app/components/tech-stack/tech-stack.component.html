Real Time Analytics
A module to visualize data from various business and healthcare perspective where a provider or a supplier can see and make some great reports to check how their patients, services or devices can be made better for the overall wellness.
RaspberryPi
It is now one of the most popular technology items in the world. It can function as a proper desktop computer or be used to build smart devices.
API Gateway
A gateway to connect with Cockpit to transfer data between facilitator and provider databases to Cockpit Consoles for data management, provider management, report generation and various monitoring use cases. APIGateway will communicate with IOTGateway via Kafka to receive data from various devices.
Cockpit
A complete web based solution for Welloculus facilitators, healthcare providers and device suppliers where each user can perform various operations based upon their roles and manage their association with Welloculus effectively to make the most use of it.
IOT Gateway
A gateway to connect with multiple health monitoring devices via Raspberry Pi or mobile application. The gateway will receive data from various devices and channels and transfer it to relevant endpoint for further action.
Mobile App
